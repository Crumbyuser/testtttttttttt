Website/AI Creation and AI Integration

Technologies: HTML, CSS, JavaScript
Duration: 3 months
Role: Full Stack Developer
View on GitHub

This project is a modern, responsive medical website designed to demonstrate the integration of advanced AI chatbot technology for providing medical advice. The site features four distinct versions: two powered by Gemini and two by OpenAI, each offering unique conversational experiences. Users can interact with the AI chatbot to receive medical guidance, explore a clean and professional interface, and navigate through user dashboards tailored for both Gemini and OpenAI implementations. The project showcases expertise in web development, user experience design, and the practical application of AI in healthcare.
